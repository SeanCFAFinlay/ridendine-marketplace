# Page Blueprints

Blueprints are available in the customer web app under `/ui-blueprint/[screen]`.

Customer screens: `customer-home`, `customer-menu`, `customer-checkout`, `customer-order-tracking`, `customer-account`.

Chef screens: `chef-dashboard`, `chef-orders`, `chef-menu-manager`, `chef-analytics`, `chef-settings`.

Driver screens: `driver-home`, `driver-offer`, `driver-active-delivery`, `driver-earnings`, `driver-settings`.

Ops screens: `ops-dashboard`, `ops-dispatch`, `ops-finance`, `ops-payouts`, `ops-reconciliation`, `ops-users`, `ops-system-health`.

Each route shows loading, empty, and unavailable state examples plus the core composition for the target page.
